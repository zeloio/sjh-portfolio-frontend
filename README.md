## Requirements

| Prerequisite      | How to check | How to install                    |
| ----------------- | ------------ | --------------------------------- |
| Node.js >= 16.x.x | `node -v`    | [nodejs.org](https://nodejs.org/) |

## Installation

-   Run `npm install` in the command line.

## Building

-   `npm run dev` to run build and start local server with files watcher
